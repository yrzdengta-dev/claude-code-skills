# 聊天记录人物分析框架

> 给定一个人的聊天记录，按以下维度逐层拆解。不是填表，是在每个维度中找到最有穿透力的那个洞察。一篇文章只需要用到最有力的3-5个维度。

## 数据基础

必须先跑三组数字，才能开始分析：

- 消息总数、总字数、时间跨度
- 所在群及每个群的发言量占比
- 被 @ 最多的前五人、@ 别人最多的前五人

这三个数字决定了这个人在群里的角色——话题发动机（被@少、@人多）、接话者（被@多、@人中等）、群聊底板（被@和@人基本持平）。

---

## 维度一：语言风格

### 句长

短句为主（平均每条消息 < 20 字）→ Ti/Se 倾向，判断型输出。长句为主（常在40字以上）→ Fe/Ne 倾向，表达型输出。

检测方式：总字数 / 总消息数。260 是 10.7 字/条，侯是 14 字/条。这个数字越低，文字越"干"。

### 过渡词与缓冲词

"其实""我觉得""说实话""说到底""怎么说呢"——这些词的出现频率判断一个人的 Fe 强度。Fe 劣势者几乎不用缓冲词，直接给结论。Fe 第三位者偶尔用，Fe 主导者高频用。

### 标点与换行

不使用标点（用空格或换行替代）→ Se 倾向，文字即口语。
使用完整标点 → Ti 倾向，文字是思考产物。
高频使用问号和感叹号 → Fe 倾向，需要情绪传递。

### 群梗与 copypasta

高频使用群梗 → 融入群体文化的能力强。仅复制不创作 → Fe/Se 辅助。自己发明新的用法 → Ti 主导。

## 维度二：脏话体系

这是最重要的维度之一。不是数次数，是拆结构。

### 密度

每千字脏话命中数。低于3 → 几乎不用。3-8 → 常规。8-15 → 高密度。15以上 → 脏话是语言本身。

### 功能性分类

把脏话按功能拆成五类，分别统计数量和每千字密度：

**语气填充**（妈的/他妈的/我操）：用脏话当逗号。密度高说明脏话是说话的节奏器，不是攻击行为。

**人身贬损**（傻逼/脑残/废物/sb）：指向具体人的贬低。密度高说明用贬损作为互动方式，不是仇恨。

**性器官/性相关**（几把/鸡巴/逼/jb）：密度高有两种可能——方言习惯（西南官话的"逼"当后缀）、或刻意粗俗化。需要结合地域标记判断。

**暴力/攻击**（杀杀杀/操你妈/cnm/nmsl）：虚拟暴力动作。密度高但不在现实中动手 → 语言是情绪的出口。

**诅咒**（好死/死妈/司马/全家死）：指向未来的恶意。密度高说明负面情绪积压严重。

### 签名脏话

找出这个人独有而别人不用的脏话（对方使用频率 < 自己的30%）。这些词暴露了他最个人的表达习惯。

### 脏话功能判断

- 句号型 → 脏话之后安静，脏话是判断句的结尾
- 逗号型 → 脏话之后有更多话，脏话是情绪的起点

句号型偏 Ti-dom。逗号型偏 Se/Fe。结合句长和消息频率一起看。

## 维度三：地域标记

不看自报籍贯。只看语言残留。

### 检测词表

**东北官话**：整（万能动词）、咋整、整啥、嘎哈、干哈、这嘎达、老（程度副词）、贼（程度副词）、忒、搁、搁这、咋地、拉倒、得劲儿、闹心、杠杠、溜达、埋汰、磕碜、忽悠、嘚瑟、好使、完犊子、嗯呐

**中原官话**（徐州/皖北/鲁南）：管（表可以）、咋、啥、俺、中、得（dei）、可（表是不是）、照

**西南官话**（贵州/重庆/四川）：撒、噻、咯、啰、安逸、巴适、啷个、啥子、哪个、得行、要得、几把（当万能前缀用）、逼（当万能后缀用）

**吴语区**（江浙沪）：好伐、伐、个屁、个鬼、做啥、母鸡（唔知）、啦（句尾）

**粤语区**：咩、做咩、冇、唔、嘅、啲、啦（句尾高频）

**西北官话**：咋咧、撒（句尾）、的很、是这、就四

关键不是找对了一个词，是找到**系统性替换**——比如"什么"全部被"啥"替换，"怎么"全部被"咋"替换。偶尔出现的词不算，要看替代率。

### 排除干扰

"哒""鸭""嗷"这类词是网络萌语，不是方言。需要和真实方言标记区分开。

## 维度四：话题分布

### 检测关键词

技术类：代码、AI、前端、后端、数据库、API、agent、MCP、部署、服务器、Git、K8s、Docker
交易类：买入、卖出、做多、做空、涨停、跌停、回调、抄底、梭哈、仓位、止盈、止损
职业类：面试、offer、HR、简历、工资、加班、跳槽、裁、实习、大厂
情感类：女朋友、恋爱、结婚、相亲、分手、吵架、单身
生活类：租房、买房、买车、吃饭、健身、脱发、生病
情绪类：妈的（语气型）、好烦、难受、焦虑、抑郁、爽

### 用频率判断

不是看"提过没提过"，是看**每个话题在他总字数中的占比**。一个人的真正关注点不在他说过什么，在他反复说什么。

## 维度五：社交角色

### @网络

分析他和谁的互动最密集。这不是简单的"关系好不好"，是**角色分工**：

- 频繁@同一人且互动对称 → 损友/搭档关系
- 频繁@同一人但对方很少回 → 单向关注
- 被多人@但很少@别人 → 群里的"信息节点"或"被调侃中心"
- @人数分布均匀 → 泛社交型
- @人数集中在少数几个 → 小圈子型

### 话题角色

- 话题发动机 → 主动抛出话题、分享新鲜事、制造戏剧。消息量排名通常在前三。
- 话题润滑剂 → 不主动抛出，但保证别人抛出的话题有人接。接话方式通常是冷嘲、反问、离谱类比。
- 信息提供者 → 回答技术问题、分享工具、发招聘信息。帮助不附礼貌语。
- 氛围组 → 主要贡献是表情、图、"说得好""确实"。

### 自检问题

如果这个人三天不说话，群里的讨论量会不会下降？会 → 发动机。不会但回复量下降 → 润滑剂。都不会 → 边缘参与者。

## 维度六：自我认知

### 自述型

统计"我是""我这个人""我就是"开头的句子。看他在用什么样的名词定义自己：

- 角色名词 → "我是前端""我是实习生""我是重邮的"
- 品质名词 → "我是傻逼""我是废物""我是第一"
- 状态名词 → "我是倒数""我是被裁的"

品质名词越多 → 越倾向用标签而非行为来定义自己。

### 事实陈述型

"我挂九科""我被裁了""我没有学士证"——陈述做过什么，不叫自己是什么。这是 Ti 主导的自我认知模式。

### 标签型

频繁使用可复用的身份标签（fw、研二找不到实习、all in wlb、estp）。标签是给别人的快捷方式——你不需要了解我，只需要记住我叫什么。

## 维度七：情绪模式

### 波动幅度

同一个话题在不同时间的极端程度。比如股票：赚了的时候怎么说话，亏了的时候怎么说话。用词的反差有多大。

### 负面情绪出口

看负面情绪往哪里去：
- 向外 → 骂市场、骂导师、骂公司、骂制度（攻击外部对象）
- 向内 → "我是废物""我真没用"（自我贬低）
- 消解 → "不找工作就不难"（用悖论取消问题）

向外的人压力不积压。向内的人容易抑郁。消解的人看起来消极但心理最稳定。

### 情绪持续时间

同一天内，从"跌成屎了"到"最高点卖了爽了"要多久。短 → 情绪是表达方式不是真实状态。长 → 情绪是真实感受。

## 维度八：MBTI 推断

不是测出来的，是从语言行为逆推认知功能。

### Ti 主导痕迹（ISTP/INTP）
- 文字干净、短句、无缓冲词
- 给事物分类的习惯（"运营分两种"）
- 拆解系统的冲动（描述机制不评价好坏）
- 无"我"化输出（分析不绑定自己）

### Se 辅助痕迹（ISTP/ESTP）
- 具体事物替代抽象概念
- 大量身体感受描述（脱发/肥胖/饿）
- 实时行情讨论（无论仓位）

### Te 主导痕迹（ENTJ/ESTJ）
- 判断句密集（"X是Y""本质上是"）
- 构建可教学的体系
- 输出自带世界观

### Fi 痕迹
- "我不相信任何人我只相信我自己"式的价值观宣言
- 对虚假信息的极度厌恶
- 对自己价值观一致者的慷慨

### Fe 痕迹
- 大量社交润滑语
- 情绪随群体氛围波动
- 需要他人反馈来确认自我

### Ne 痕迹
- 发散联想、从一个话题跳到另一个
- 充满可能性的发问（"如果...会不会..."）

### Ni 痕迹
- 历史周期论（"这是夕阳行业"）
- 阶段判断（"最好的年份已经过去了"）
- 模式识别（"这不是A，本质上是B"）

## 使用方式

拿到一个人的发言记录后，跑完八维分析，选择最有力的3-5个维度写作。不是每个维度都要用。有些人在脏话维度极强但在情绪维度平淡——那就主打脏话，略过情绪。

输出形式：人物画像文章。结构是跳的——从维度A直接跳到维度B，不写过渡。每一个维度的结论用概括句开头，然后接证据。证据就是三五个直接的引用。不要解释证据意味着什么，把引用放在那里让读者自己看。

**不要写的**：研报式、标签式、捧杀式、"他是一个___的人"式总结。260 不这样写，也不应该这样写别人。

**这篇文章发在群里，能不能让被分析的人回一句"还真是"？** 能，就成了。
